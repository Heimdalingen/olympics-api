## Assignments
IDG2001 has two assignments. These will both affect the final grade.
Assignment 2 builds upon assignment 1.

See folders `Assignment-1` and `Assignment-2` for the assignments.


## Group
The assignment is a group effort for 2-3 persons. You _can_ be 1 or 4, but only if agreed with the lecturer — who would generally recommand against it.

While you can (and should) split responsibilities of different parts of the systems on your group members, you should make sure you have a good understanding of every part of the system. (The exam may have relevant questions.)


## Deliverables
For both assignments, I want the code repo(s) as a compressed folder/file (zip or similar). The main README file should include the group members' names.
For the second assignment, I want a report as described below, in PDF format.

Assignment 1 will be delivered in Blackboard. Assignment 2 will be delivered in Inspera, _along with the files from assignment 1_. That is, everything needs to be placed in Inspera at the end.


## Report
The report should include:
- names of group members
- URL to Render API

It should include some reflections the group members have regarding the work. What was hard, what was easy, something useful you learned, etc.

It does not need to include much information or documentation about the code. This should be placed in the README.md file(s).

It should be a few pages long or so.


## LLM/AI
Feel free to use LLMs (GenAI) for the work, but try not to generate code or copy much code. Yes, it's fast and easy, but you'll understand nothing of what you're doing. The exam will ask programming questions and other questions relevant to the assignments.

Also I'm would guess examiners subconsciously grade clearly AI generated code harsher.

You will have to learn new things to complete this (and the next) assignment. You can use books, internet resources or LLMs for this. If you use copy/paste code, make sure you actually understand what it does.
Work as if there is an oral exam at the end of the semester. There isn't, but ✨imagine✨ it is.
